# 🔗 Blockchain Smart Contract Project

A decentralized application (DApp) built using **Blockchain technology and Solidity smart contracts**.
This project demonstrates how smart contracts can automate transactions securely without intermediaries.

---

## 🚀 Features

✔ Smart contract development using **Solidity**
✔ Secure and transparent blockchain transactions
✔ Automated execution of contract conditions
✔ Decentralized architecture
✔ Tested using local blockchain environment

---

## 🛠 Tech Stack

* **Solidity** – Smart contract programming
* **Ethereum Blockchain** – Decentralized network
* **Remix IDE / Hardhat / Truffle** – Development & testing
* **JavaScript** – Interaction with smart contract
* **MetaMask** – Wallet integration

## 🔍 How It Works

1. A smart contract is written in **Solidity**.
2. The contract is compiled and deployed to a blockchain network.
3. Users interact with the contract using a wallet like **MetaMask**.
4. The contract automatically executes predefined rules.
